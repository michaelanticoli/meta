# Project Images

Please add your project images to this folder.

## Required Images

Refer to the corresponding MDX file in `/src/app/work/projects/` to see which
image filenames are expected.

## Image Guidelines

- **Formats:** PNG or JPG
- **Aspect Ratios:**
  - Hero/cover: 16:9 or 3:2
  - Gallery tiles: 1:1 or 4:3
  - Detail shots:  Consistent per section
- **Naming convention:** Use descriptive, lowercase names with hyphens
  - Example: `campaign-hero. png`, `social-media-assets.png`

## Placeholder Images

You can use placeholder images initially from https://placeholder.com or
https://unsplash.com and replace them with actual project images later.

To generate a placeholder:
- https://via.placeholder.com/1920x1080.png? text=Project+Hero
- https://via.placeholder.com/800x800.png?text=Gallery+Image
